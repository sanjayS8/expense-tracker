# 💰 Expense Tracker

A modern, fully functional expense tracking web application that helps you manage your finances smartly.

## 🌐 Live Demo
**Access online:** https://sanjayS8.github.io/expense-tracker/

Works on:
- 📱 **Mobile** - iPhone, Android, and all devices
- 💻 **Laptop** - Desktop browsers (Chrome, Firefox, Safari, Edge)
- 🌍 **Any device** with internet access

## ✨ Features

### 💸 Core Functionality
- ➕ **Add Expenses** - Record expenses with amount, category, description, and date
- ✏️ **Edit Expenses** - Modify any expense anytime
- 🗑️ **Delete Expenses** - Remove unwanted entries
- 💾 **Auto-Save** - Data saves automatically in browser (persists even after closing)

### 📊 Analytics & Statistics
- 📈 **Real-time Statistics** - View total, average, highest, and today's expenses
- 📅 **Monthly Breakdown** - See category-wise expenses for current month
- 📊 **Interactive Pie Chart** - Visual representation of spending by category
- 📋 **Category Filters** - View expenses by specific categories

### 🎯 Categories
- 🍔 Food & Dining
- 🚗 Transport
- 🎬 Entertainment
- 💡 Utilities
- 🏥 Healthcare
- 📌 Other

### 💱 Currency & Format
- ₹ **Indian Rupees** - All amounts in INR
- 📐 **Proper Number Formatting** - Easy to read (e.g., ₹10,000)
- 🌍 **Localized Dates** - Date format suitable for India

### 🎨 Design
- 🌈 **Modern UI** - Beautiful gradient design with smooth animations
- 📱 **Fully Responsive** - Works perfectly on mobile and desktop
- ⚡ **Fast & Lightweight** - No server needed, all processing is client-side

## 🚀 How to Use

1. **Open the app:** https://sanjayS8.github.io/expense-tracker/
2. **Add an expense:**
   - Enter amount in rupees
   - Select a category
   - Add description/details
   - Choose the date
   - Click "Add Expense"
3. **Track your spending:**
   - View real-time statistics
   - Check category breakdown
   - Filter by categories
4. **Edit or delete:** Use the Edit/Delete buttons on each expense

## 💡 Features Explained

### Today's Summary
Quickly see how much you've spent today without scrolling through all entries.

### Monthly Breakdown
Understand your spending patterns for the current month by category.

### Statistics Dashboard
- **Total Expenses** - Sum of all your expenses
- **Total Items** - Number of transactions recorded
- **Average Expense** - Mean expense amount
- **Highest Expense** - Maximum single expense

### Pie Chart
Visual representation helps you understand which category consumes most of your budget.

### Filters
Quickly find expenses by clicking on category buttons. "All" shows everything.
